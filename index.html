What we are designing

TwinHome is a Software As a Service (SaaS) which can be accessed using a mobile phone / web based interface that allows users to import a 3d Model of a physical space, invite others to collaborate in that model. Examples of collaboration could be for locating / finding objects in the physical space via the virtual copy, share methods of organizing and manipulation of the physical environment via virtual copy for training and remote collaboration purposes.
